关于python以及anaconda环境安装

Created2022-10-14|Updated2022-10-04|编程python
|Post View:

关于python

python，英文名为蟒蛇，下图于广州动物园拍摄

关于为什么叫这样的名字俺也不知道。总之，这是一门编程语言。这门语言最大的特点便是：简单，上手十分容易。特别适合快速完成零散的任务。

python版本

python的版本首先分为两个大版本：python2和python3。这两个版本之间互不兼容。不过呢，不会吧不会吧不会都2202年了还有人在用python2吧~~事实上python2已经逐渐地退出了历史舞台，我们使用的无一例外都是python3

python3也分为很多小版本，目前最新的是python3.10，各个python版本在电脑上是可以共存的，然而你肯定不想在记住有多少个版本，所以呢，我们使用Anaconda来帮助我们管理python的版本和各种包

关于Anaconda

anaconda是什么？anaconda是水蟒。。。

好了，anaconda是一个管理器，它可以管理python的版本和各种python的包。所谓包，可以理解为“小零件”，单独的python不会将所有的功能放在一起，那样程序的体积就会无比巨大臃肿。所以，他将不同的功能打包成各种不同的包，需要某个功能的时候就引入对应的包，就是这样~

Anaconda的安装

  • tips：安装anaconda后不需要安装python，anaconda里面已经有了

下载安装anaconda

贴一个官网下载链接：Anaconda | Anaconda Distribution

  • 安装软件的时候建议右键，以管理员身份运行
  • 安装时选择的安装目录请不要出现中文！
    • 错误的：D:\真不错\anaconda\
    • 正确的：D:\zbc\anaconda\

Vscode配置python环境

哎？不是该安装的已经安装完毕了嘛？是的！但是呢，为了让敲代码有良好的体验，还是建议下载一个vscode。

vscode是什么

vscode是一个代码编辑器，通俗的说，是一个高级记事本，只不过，这个记事本可以安装很多插件使得他的功能十分强大。安装某些插件可以让你在vscode里面直接运行代码（事实上这也是他的一个主要用处）。而且他十分轻便，打开迅速，对于轻薄本笔记本电脑十分友好（这里点名批评pycharm）。还有最最重要的一点是：界面好看😂

安装vscode

贴一个下载链接：Visual Studio Code - Code Editing. Redefined

安装没什么好说的，一路yes，注意为保险起见路径不要有中文就ok了

安装python插件

插件安装方法：打开vscode，点击左侧菜单栏最后一个图案是四个正方形的按钮，搜索要安装的插件就可以了

首先建议安装中文插件：Chinese (Simplified) (简体中文) Language Pack for Visual Studio Code

再进入正题，搜索python并安装第一个插件，这样，你的python环境就安装完毕了，简单！
